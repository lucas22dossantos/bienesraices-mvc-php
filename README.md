# Bienes Raíces MVC (PHP)

Proyecto desarrollado como parte de un **curso de PHP y arquitectura MVC**, donde se aplica **programación orientada a objetos (POO)** y buenas prácticas de desarrollo web.

Este proyecto tiene como objetivo aprender a estructurar una aplicación en **PHP** utilizando el patrón **MVC (Modelo - Vista - Controlador)** y conectarla a una base de datos **MySQL**.

---

## Características principales

Estructura basada en el patrón **MVC**  
 Código modular y reutilizable con **clases y objetos**  
 CRUD completo (crear, listar, actualizar y eliminar)  
 Validación de datos del lado del servidor  
 Conexión a base de datos **MySQL**  
 Automatización de tareas con **Gulp**  
 Separación entre lógica, vistas y modelos

---

## Tecnologías utilizadas

PHP 8+
MySQL
Gulp
HTML5, CSS3, JavaScript
Node.js / npm

---

## Sobre el proyecto

Este proyecto forma parte de un **curso de PHP orientado a objetos y MVC**, realizado con fines educativos para reforzar conceptos de arquitectura, estructura de carpetas y desarrollo web moderno.

El enfoque principal es comprender cómo separar la lógica de negocio (modelos), la lógica de control (controladores) y la presentación (vistas).
